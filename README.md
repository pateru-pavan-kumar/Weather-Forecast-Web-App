# Global Weather Dashboard

A fully responsive, real-time weather application built for non-commercial use.

## Key Features
* **Real-time Data:** Fetches current weather and a 3-day forecast using the OpenWeatherMap API.
* **Dynamic UI:** Background animations (rain, clouds, clear sky, night mode) synchronized with the current weather condition and time of day.
* **Technologies:** Pure HTML5, JavaScript (ES6+ for asynchronous API calls), and Tailwind CSS for rapid, responsive styling.
